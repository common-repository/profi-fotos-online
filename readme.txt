== Description ==

* __kostenloser, direkter Support per Telefon, WhatsApp und E-Mail__
* Fotografen-Online-Shop mit direkter Anbindung an unsere Fachlabore www.pixelfoto-express.de und www.expressphoto.de.
-- Viele, unterschiedliche Foto-Formate: Fotos, Poster, Wandbilder, Sticker, Passbilder, Postkarten, Tassen, Plüschtiere, Puzzles und vieles mehr.
-- Unterschiedliche Preis-Profile möglich.
-- Direkter Versand aus dem Labor an Deine Kunden im neutralen Umschlag.
-- Sammelbestellung zum Sparen von Versandgebühren.
-- Geeignet für Schul- und Kindergartenfotografie, Abi-Bälle und Schul-Abschlussfeiern, Sportturniere.
-- Anbindung eines Fachlabors in Österreich ist aktuell in der Umsetzung und Anfang 2023 verfügbar.
* Online-Shop zum Generieren von Bestell-Listen (zur Weiterverarbeitung über Dein eigenes Fachlabor).
* Anbindung des eigenen Fachlabors auf Anfrage.
* Rabatt-Einstellungen möglich (absolute oder prozentuelle Rabatte).
* Versandkostenfrei-Einstellung ab einem individuell festgelegten Limit.
* Spar-Pakete.
* Alben mit unterschiedlichen Zugangsmöglichkeiten:

-- öffentliche Alben.
* Shootings / Albumverwaltung mit QR-Codes.
-- Automatisiertes Erstellen von Alben und Einsortieren der Fotos aus einem Gruppen-Album mit QR-Codes.
-- Zugangskarten für Kunden mit individuellem Zugang zu den Einzelfotos.
* Downloads anbieten / ZIP-Download.
* Zahlungsabwicklung (PayPal, SOFORT-Überweisung, SEPA).

== Installation ==

Plugin im WordPress Backend suchen:
Suchbegriff: <code>profi-fotos-online</code>
und automatisch installieren.
- oder -
das Zip-File entpacken und per FTP in das Verzeichnis /wp-content/plugins/ kopieren.

Das Plugin muss im WordPress Backend aktiviert werden.

Du musst ein gültiges Konto auf https://www.profi-fotos-online.com/ haben und dort mindestens ein Album angelegt haben. Deine WordPress-Seite muss unter https://www.profi-fotos-online.com/ eingetragen werden.
Danach kann der vollständige Shop über den Short Code [pfo] bei Dir eingebunden werden.

Nach der Einbindung des Short Codes werden etwaige Probleme erfasst und auf der Webseite angezeigt. Achte dort bitte auf die Hinweise.


== Frequently Asked Questions ==

= Ist die Nutzung des Plugin kostenlos =
Ja, die Nutzung ist kostenlos, ebenfalls kann das Plugin mit einem kostenlosten Konto auf https://www.profi-fotos-online.com/ genutzt werden.
= Wann fallen Gebühren an? =
Gebühren fallen nur dann an, wenn Kunden über den Shop Fotos bestellen oder downloaden. Die Galerien und die Nutzerverwaltung für Deine Kunden sind kostenlos. 

== Screenshots ==

1. Hier ist das Plugin mit dem Short Code [pfo] eingebunden. Login-Formulare, Alben-Auswahl usw. werden automatisch, abhängig von Deinen Alben-Einstellungen, angezeigt.
2. In der Alben-Übersicht können sich Deine Kunden einen Überblick über die Fotos verschaffen. Je nach eingestelltem Layout sortieren sich Quer- und Hochformate automatisch an.
3. In der Bild-Detailansicht können die Kunden zwischen unterschiedlichen Formaten auswählen. Welche Formate angezeigt werden, legst Du selbst fest.
4. Über die Kopf-Leiste kann zu den anderen Alben gewechselt werden. Im Warenkorb werden die ausgewählten Fotos und Formate angezeigt und können noch variiert werden.
5. In der Verkaufsabwicklung geben die Kunden Ihre Login- und Adressdaten ein.
6. Die Kunden können zwischen 3 Bezahlarten auswählen, aber auch nach Abschluss der Bestellung eine andere von den 3 Arten nutzen. Die Gebühren für die Zahlungsabwicklung tragen wir.
7. Am Ende der Bestellung erhalten die Kunden eine Bestellbestätigung sowie die Links zu den Payment-Partnern.

== Known Issues ==

Inkompatibilität mit Caching-Plugins, z.B. WP Super Cache

* Je nach Cache-Plugins werden die Seiten der Bestell-Abwicklung gecacht. Eine Bestellung kann dann nicht fehlerfrei durchgeführt werden.

Zwei wesentliche Einstellungen müssen u.U. vorgenommen werden:

* Sessions generell aktivieren
* jQuery für Plugin aktivieren

Beide Einstellungen werden nach der Installation geprüft und beim ersten Aufruf der Vorschau angezeigt. Folge den Hinweisen auf der Webseite. Diese Einstellungen werden im WordPress-Backend eingestellt.

Weitere Voraussetzungen  bzw. aufgetretene Schwierigkeiten

* Es müssen Sessions auf dem Webserver / Hosting-Paket aktiviert sein.
* Beim Öffnen der Plugin-Dateien mit unterschiedlichen Editoren konnten wir beobachten, dass der Zeichensatz der Datei automatisch konvertiert wurde. Die Dateien müssen alle UFT8 ohne BOM codiert sein.
* Themes, die nicht fachgerecht bearbeitet wurden können dazu führen, dass unser Plugins nicht funktioniert.
